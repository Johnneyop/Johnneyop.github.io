Sleep Tracker App

The sleep tracker app records the time and quaility of sleep. It stores all this information in a SQL database. This app illustrates the following programming practices:

- Using SQLite to manipulate data in database room. By creating custom methods with the following functions (Insert, delete, update, query, etc...).

- Using multi-threading or coroutines to improve speed of app.

- Utilize Live Data and navigatation classes to handle data between UI display and database.
