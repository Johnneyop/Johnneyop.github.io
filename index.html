Guess The Word App

This is a Charades game app. It generate words, keep track of scores, and displays a timer. This app illustrates the following programming practices:

- Making apps in MVVM architecture which uses: Fragments, view models, and view model factory files

- Implementation of Live Data to make data lifecycle aware

- The importance to keep apps modular to make it easy to test/ debug

Sleep Tracker App

This sleep tracker app lets you record how long was your sleep and rate the quality of it. It stores all this information in a SQLite database. This app illustrates the following programming practices:

- Using SQLite to manipulate data in database room. To handle data, the app uses custom methods in data access object file. (Insert, delete, update, query)

- Using multi-threading or coroutines to reduce workload on main method and to make app run smoother.

- Using buttons to handle Live Data and navigate to different fragments.

Real Estate App

This real estate app display lands that you can buy or rent in Mars for housing properties. The app pulls information from a server and filters it for users. This app illustrates the following programming practices:

- The use of RESTful api and retrofit library to request and get data from servers.

- The use of moshi library to parse JSON strings into Java / Kotlin objects.

- The use of glide library and binding adapters to load and display images on the app. Also uses recycler view to process the images smoothly while scrolling.

- Created a menu to filter lands for sale or rent and list their prices.
